# LVL UI Repository

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

### Install Node Modules

To install the node modules necessary, run: 

###### NPM:

```shell
npm install
```

**OR**

###### Yarn:

```shell
yarn install
```

## Running the UI

To run UI against local API, run:

###### NPM:

```shell
npm start
```

**OR**

###### Yarn:

```shell
yarn start
```
